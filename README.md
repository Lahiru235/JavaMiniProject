# JavaMiniProject

A simple Java mini project built using OOP concepts, featuring core functionalities and clean code structure for academic purposes.

## 📌 Overview

This is a **Learning Management System (LMS)** application built with Java and JavaFX. 
It demonstrates fundamental OOP principles with a GUI interface and MySQL database integration.

## 🛠 Tech Stack

- **Language**: Java 17
- **GUI**: JavaFX 21.0.6
- **Database**: MySQL 9.6.0


✨ Features
Object-Oriented design pattern
JavaFX GUI interface
MySQL database connectivity
Unit testing with JUnit 5
Clean code structure
